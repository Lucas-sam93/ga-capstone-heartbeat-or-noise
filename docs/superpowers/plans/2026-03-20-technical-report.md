**Goal:** Create the three capstone deliverables — technical report notebook, DOCX report, and Streamlit dashboard — documenting all existing research results.

**Architecture:** Notebook generates all 15 figures and contains the full narrative. DOCX script builds on the existing `outputs/generate_docx.py` pattern to produce a code-free version. **Tech Stack:** Python 3.13, matplotlib, seaborn, pandas, numpy, scikit-learn, joblib, python-docx, streamlit, plotly

**Important rules (from CLAUDE.md):**
- Do NOT include `Co-Authored-By` trailers in any commit message
- Do NOT call `fit_transform()` on any non-training data — `transform()` only
- Do NOT retrain any model — all figures use pre-computed results
- Do NOT modify the fixed threshold (0.34) when evaluating Layer 2
- All figures saved to `outputs/figures/` and logged in `figures_log.csv`
